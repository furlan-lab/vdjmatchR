# Locate packaged VDJdb file bundled with vdjmatchR

Locate packaged VDJdb file bundled with vdjmatchR

## Usage

``` r
vdjdb_packaged_path(use_fat_db = FALSE)
```

## Arguments

- use_fat_db:

  logical; TRUE for full db if bundled, FALSE for slim (default)

## Value

string file path (empty string if not found)
