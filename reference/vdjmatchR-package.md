# vdjmatchR: R Interface to VDJmatch (Rust)

R bindings to query TCR sequences using a Rust implementation derived
from vdjmatch-rs, via extendr.

Provides fast TCR matching against VDJdb-formatted databases using a
Rust engine exposed via extendr. Bundles slim/full sample databases and
supports using user-supplied database files.
