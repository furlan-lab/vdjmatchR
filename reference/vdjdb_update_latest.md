# Update (download) the VDJdb file and return its path into extdata

Update (download) the VDJdb file and return its path into extdata

## Usage

``` r
vdjdb_update_latest(use_fat_db = FALSE)
```

## Arguments

- use_fat_db:

  logical; TRUE for full db, FALSE for slim (default)

## Value

string file path to the updated file
